# 🚀 Keven Wendell — Portfolio

Este é o meu **portfólio profissional**, desenvolvido em **React.js** e hospedado no **GitHub Pages**.  
O projeto apresenta meus **principais trabalhos, habilidades e projetos em desenvolvimento**, com foco em **interfaces limpas, responsivas e funcionais**.


## 🧠 Tecnologias Utilizadas

- ⚛️ **React.js** — Criação de componentes reutilizáveis e dinâmicos  
- 🎨 **CSS3 / Tailwind (ou CSS puro)** — Estilização e responsividade  
- 🌐 **Vite** — Build rápido e leve para ambiente React  
- 📦 **GitHub Pages** — Hospedagem gratuita e prática  


## 💼 Funcionalidades

- Sessão **“Sobre Mim”** com resumo da minha trajetória e objetivos  
- Seção **“Projetos”** com status (em desenvolvimento, concluído, etc.)  
- **Design responsivo** e moderno  
- Integração com **GitHub e LinkedIn**  
- Interface leve e otimizada para desempenho
